## Prerequisites
ハンズオンを実施するにあたり、以下が必要です。

- Datadogアカウント
  - [フリートライアル](https://www.datadoghq.com/ja/free-datadog-trial/)へのサインアップ、もしくは有償のアカウントが必要です。Freeプランでは実施できません。
- AWSアカウント
  - EC2インスタンスを作成、接続するための設定を行う権限が必要です。
  - AWS上でインターネットに接続できるVPCが必要です。
